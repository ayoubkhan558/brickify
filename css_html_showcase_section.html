Documentation

Getting started
Install the CLI with npm install -g @atlas/cli then run atlas init. Takes under 60 seconds on a fresh repo. See the quickstart guide for details.
Authentication
Atlas uses OIDC with PKCE. Tokens expire after 1 hour and refresh silently. For machine-to-machine, generate a service account token in settings.
Webhooks reference
All events POST a JSON body signed with HMAC-SHA256. Verify the X-Atlas-Signature header before processing. Retries follow an exponential back-off — up to 5 attempts over 24h.
